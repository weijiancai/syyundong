赛事介绍

    2014年全国定向冠军赛是由国家体育总局航空无线电模型运动管理中心、中国定向运动协会、江苏省体育局主办，宿迁市人民政府、省无线电和定向运动协会具体承办，智美集团为商业运营机构的国家级规模较大、影响力较广的大型赛事活动。比赛将于2014年10月29日-31日在江苏省宿迁市举办，届时将有来自全国各地的近500名运动员参加9个组别的比赛。

定向大赛，即一种以定向运动为方式的比赛项目。
定向运动是一项非常健康的智慧型体育项目，是智力和体力并重的运动。它不仅能强健体魄，而且能培养人独立思考，独立解决问题的能力及在体力和智力受到压力下做出迅速反应、果断决定的能力。
定向运动是一项学生体育项目，定向越野就是运动员借助地形图和指北针按规定的顺序独立完成寻找若干个标绘在地图上的地面检查点，并以最短的时间跑完全赛程的运动。它可训练人们识图、用图能力、野外定向能力，并锻炼人们的奔跑能力。其是一项娱乐性、实用性极强的体育运动项目，尤其可以提高成年人的野外生存与团队合作能力。它分为五日定向、专线定向、计分定向、夜间定向、滑雪定向、接力定向等。
